Esse projeto é um site memória criado para registrar minha primeira viagem à Serra Negra, no interior de São Paulo.
A ideia desse site é funcionar como um espaço digital de lembranças, unindo a história da cidade, roteiro turístico e registros visuais da experiência.

A proposta é transformar uma vivência pessoal em um projeto web com identidade, sensibilidade e cuidado estético.

O site conta a historia da cidade, sua relação com as águas mineirais, o desenvolvimento como estãncia turística, lugares que marcaram a viagem e os momentos registrados em fotos.

Estrutura do site

Sobre
  Apresenta o contexto da viagem.

A cidade
  Apresenta um breve relato sobre a origem de Serra Negra, suas águas mineiras e sua identidade como estância turística.

Pontos Turísticos
  Apresenta um guia dos principais pontos turísticos da cidade.

Galeria
  Um espaço visual dedicado às fotos da viagem, onde as imagnes falam por si.

Técnologias utilizadas:

- HTML5
- CSS3

Conceitos aplicados
- HTML semântico
- Storytelling aplicado ao design web
- Hierarquia visual
- Grid layout e ritmo visual

Esse projeto foi criado para particar HTML e CSS de forma consciente, explorar narrativa visual em pãginas estáticas, demonstrar atenção a detalhes de laytou e conteúdo.
